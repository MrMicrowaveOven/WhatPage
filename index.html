Welcome to What Page Am I On? This app can be used to tell you the rank of a site's Google's search results.

For example, if you search "custom email" with keyword "iwantmyname", it will display a 2 because "iwantmyname.com" is the 2nd result of that Google search.

Note that the number might be off from a manual search, since a manual search involves cookies, location, etc.
