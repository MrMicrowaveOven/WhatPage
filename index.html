Welcome to What Page Am I On? This app can be used to tell you what page of Google's search results a certain site is on.

With the current version, it will tell you which result of Page 1 your current keyword is contained in. For example, if you search "custom email" with keyword "iwantmyname", it will display a 2 because "iwantmyname.com" is the 2nd result of that google search.

Later versions will allow you to search further pages of google, but right now it can only search the first page. Note that the number might be off from a manual search, since a manual search involves cookies, location, etc.
