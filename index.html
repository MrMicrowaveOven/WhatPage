Welcome to What Page Am I On? Given a set of search terms and a website url, this app returns the rank of the site in Google's search results.

For example, using the search terms "social network" with url "facebook.com" will result in a 73 because "facebook.com" is the 73th result of that Google search.

Note that WHAT PAGE AM I ON results may differ slightly from a manual search, since a manual search involves cookies, location, etc.
